# 📦 Telegram Channel Parser Bot

Бот собирает последние 20 постов из любого публичного Telegram-канала и отдаёт их в виде JSON-файла прямо в чат.

---

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Получение ключей

**API_ID и API_HASH** (нужны один раз для MTProto):
1. Зайди на https://my.telegram.org
2. Войди под своим номером телефона
3. Перейди в «API development tools»
4. Создай приложение → скопируй `App api_id` и `App api_hash`

**BOT_TOKEN**:
1. Напиши @BotFather в Telegram
2. Команда `/newbot` → следуй инструкциям
3. Скопируй токен вида `123456789:AABBcc...`

### 3. Настройка .env

```bash
cp .env.example .env
# Открой .env и вставь свои значения
```

### 4. Запуск

```bash
python bot.py
```

## 💬 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/parse @channel` | Собрать 20 последних постов |
| `/parse https://t.me/channel` | То же по ссылке |
| `/help` | Справка и формат JSON |

## 📄 Структура JSON-файла

```json
{
  "channel": "durov",
  "channel_id": 476038099,
  "fetched_at": "2026-03-14T12:00:00.000000",
  "total_posts": 20,
  "posts": [
    {
      "id": 1234,
      "date": "2026-03-14T10:30:00+00:00",
      "text": "Текст поста...",
      "views": 150000,
      "forwards": 3200,
      "replies": 87,
      "media_type": "photo",
      "edit_date": null,
      "grouped_id": null
    }
  ]
}
```

### Поля поста

| Поле | Описание |
|------|----------|
| `id` | ID сообщения в канале |
| `date` | Дата публикации (ISO 8601) |
| `text` | Текст поста |
| `views` | Количество просмотров |
| `forwards` | Количество пересылок |
| `replies` | Количество ответов |
| `media_type` | `photo`, `video`, `audio`, `document` или `null` |
| `edit_date` | Дата последнего редактирования |
| `grouped_id` | ID медиагруппы (если пост — часть альбома) |

## ⚠️ Важно

- Бот работает только с **публичными** каналами
- Для приватных каналов нужно, чтобы аккаунт, выдавший `API_ID/HASH`, был подписан на канал
- Файл `bot_session.session` — кэш сессии Telethon, не удаляй его во время работы
